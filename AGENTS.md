# Project instructions

Project: Evropa Sad public website redesign

Stack:
- Astro
- TypeScript
- Tailwind CSS
- Static JSON mock data for now
- Later API integration with existing custom CRM/platform

Design direction:
- Modern institutional political website
- Navy, blue, white and gold palette
- Serious, European, trustworthy, results-focused
- Avoid playful startup style
- Avoid generic WordPress-looking layouts

Frontend rules:
- Use Astro components
- Use Tailwind utility classes
- Mobile-first responsive design
- Semantic HTML
- Accessibility-first
- Reusable components
- Keep pages clean and modular
- Avoid inline styles
- Avoid unnecessary JavaScript
- No external UI component libraries for now

Content sections:
- Homepage
- Results
- Program
- Organization
- News
- Documents
- Contact

Organization page must support:
- Predsjedništvo
- Glavni odbor
- Izvršni odbor
- Nadzorni odbor
- Statutarna komisija
- Lokalni odbori
- Individual person profile cards

Data:
Use mock JSON files inside src/data until API integration is added.

Naming:
- Components use PascalCase
- Astro pages use lowercase slugs
- Keep CSS mostly in Tailwind
